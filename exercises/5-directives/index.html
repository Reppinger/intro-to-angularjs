Directives

  1. Create a controller on the `app` module called `HoverController`, that has a message property assigned to the `$scope`. Let the message read: "Mouse Over these images to see a directive at work!"
  2. Assign the controller to div#hover-interaction using `ng-controller`
  3. On each house image element in the page, add a `shows-message-when-hovered` attribute and a `message` attribute. Make the `message` attribute value a string: "Im the nth house", where nth is `first`, `second`, etc..
  4. Create an `attribute` directive within the `app` module called `showsMessageWhenHovered` and add a `link` function that uses `angular.element.bind` to toggle the `scope.message` on 2 events: `mouseenter` and `mouseleave`.
  5. Create a binding for the message on the div.alert-box element using `ng-bind` to see the message changing when you mouseover the images on the page.

Note: Open your browsers developer tools console and make sure the tests are passing.
